# Archie's Magnificent and Mysterious Maze V2.0

This is a map set for [Hideous Destructor](https://codeberg.org/mc776/HideousDestructor).

It also Requires the [Viridian Firing Range](https://github.com/Slogstin/HD_Viridian_Range_-Vanilla-) For a bunch of textures

The objective is quite simple: Take out all the Archies/Necromancers to unlock the exit. Simple, right?

No, the music is not broken. It just plays when you open the door.

Difficulties change the number of Archviles/Necromancers on the map. (Hideous and Beautiful have the same number though.)

There are Co-op spawns and Co-op exclusive monsters & items. (Someone please test this map in Co-op, so i know if it's good or too easy.)
